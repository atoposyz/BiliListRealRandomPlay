# BiliListRealRandomPlay
让b站的收藏夹随机播放变成随机播放

[安装](https://github.com/atoposyz/BiliListRealRandomPlay/raw/refs/heads/main/RandomPlay.user.js)

## 展示

在收藏夹的播放页面中添加了一个按钮

其中▶表示当前启动了脚本的随机播放，⏸表示关闭了脚本的随机播放

## 功能

在启动后，脚本会请求当前收藏夹所有视频并打乱顺序，按打乱后的顺序**跳转网页**并**自动播放**

**注意**：收藏夹必须为公开状态！
